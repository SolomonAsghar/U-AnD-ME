# U-AnD-ME
U-Net 3+ for Anomalous Diffusion analysis enhanced with Mixture Estimates. 

# Usage instructions
1) Clone using Git LSF.
2) Use Prediction/Predict_GeneralNet.ipynb for predictions using the phase one general network, use Prediction/Predict_ExpNets.ipynb for predictions using phase two experiment specific networks (note, these will only work well on the AnDi Challege dataset).
3) Ensure data_path points to the correct data folder.
4) Run all cells in the notebook. The output (in AnDi format) will be in a folder in Prediction.
